# 시작

## 필요한것

개발 언어 : Python 3.x , html, css, javascript

개발 툴 : Pycharm, Atom etc

Python 필요 모듈 : Django, Pillow

> django 설치
>
> $ pip3 install django
>
> pillow 설치
>
> $ pip3 install pillow



## 실행 방법

$ cd LetsGo

$ python3 manage.py runserver

> 백그라운드 실행
>
> $ python3 manage.py runserver &
